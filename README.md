# SoftCodec - Premium Digital Solutions Agency

SoftCodec is a high-end SaaS and digital solutions agency website built with a focus on premium aesthetics, interactive 3D visuals, and seamless user experiences.

## 🚀 Core Features
- **Modernized Service Pages**: Comprehensive overhauls for Web Development, AI Automation, Cloud/DevOps, UI/UX Design, E-commerce, and more.
- **Premium Design System**: Unified "Glassmorphism" aesthetic with high-fidelity 3D assets and dynamic motion loops.
- **Interactive Animations**: Powered by `framer-motion`, featuring floating hero assets, dynamic floor shadows, and scroll-triggered reveals.
- **Responsive Architecture**: Mobile-first design engineered to scale from handheld devices to enterprise 4K dashboards.
- **High-Performance Stack**: Built on Next.js 15 for absolute speed and optimized rendering.

## 🛠️ Technology Stack
- **Framework**: [Next.js 15+](https://nextjs.org/)
- **Styling**: [TailwindCSS](https://tailwindcss.com/) & [Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Typography**: Orbitron & Inter (Google Fonts)

## 📂 Project Structure
```text
app/
├── services/          # Modernized service-specific pages
├── components/        # Reusable UI components (Hero, Stats, WorkProcess, etc.)
├── layout.tsx         # Global layout and theme management
└── page.tsx           # High-impact home landing page
public/               # High-fidelity 3D assets and branding imagery
```

## ⚡ Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Locally**:
   ```bash
   npm run dev
   ```

3. **Production Build**:
   ```bash
   npm run build
   npm run start
   ```
